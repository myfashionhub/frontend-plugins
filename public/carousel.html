This a simple carousel that runs on a schedule

About the carousel

This light-weight carousel (2K) allows you to easily create a home page slider that runs on a schedule.

  • Customize the dates you want each slide to play.
  • Specify the speed of the carousel
  • Optimize load time with lazy load (On initial page load, it loads only the first image instead of all the images at once)
